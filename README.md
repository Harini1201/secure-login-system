# secure-login-system
Secure Login System is a web-based authentication application developed using Python Flask. The project enables users to register and log in securely by storing passwords with bcrypt hashing instead of plain text. It includes basic input validation, protection against SQL injection through parameterized queries, session management for authenticated users, and a secure logout feature. The application demonstrates essential web security practices and serves as a beginner-friendly implementation of a secure user authentication system using Flask and SQLite.
